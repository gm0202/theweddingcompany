# Test Your Knowledge - Quiz App

A pixel-perfect, interactive quiz application built with Next.js 15, Tailwind CSS (v4), and Framer Motion.

## Features

- **Glassmorphism Design**: Modern, clean UI with blur effects and gradients.
- **Interactive**: Smooth transitions between questions using Framer Motion.
- **Responsive**: Optimized for desktop (as per requirements) but mobile-friendly.
- **State Management**: Tracks user progress and specific answers.
- **Result System**: Calculates final score.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the app**:
   Visit [http://localhost:3000](http://localhost:3000)

## Project Structure

- `src/app/page.tsx`: Main quiz logic and state.
- `src/components/`: Reusable UI components (ProgressBar, OptionButton, ResultCard).
- `src/lib/utils.ts`: Utility for class merging.
- `src/app/globals.css`: Global styles and Tailwind configuration.
